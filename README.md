# 🚀 FocusFlow – Your Personalized Study Companion  

[🌐 Live Demo](https://stupendous-paprenjak-1053a3.netlify.app/)

---

## 📌 About the Project  
**FocusFlow** is a modern, all-in-one study platform designed to help students from all academic backgrounds—Engineering, Medical, Arts, Commerce, and Science—focus better, study smarter, and stay consistent.

This project started as a simple idea:  
An engineering student struggled to manage time and stay productive while juggling timers, YouTube videos, music, and subject materials. The goal was to bring all essential study tools into one seamless, personalized platform—and **FocusFlow** was born.

---

## ✨ Features  
- 🔐 Authentication System with user profile personalization  
- 📚 Field Selection: Engineering, Medical, Arts, Commerce, Science  
- ⏱️ Persistent Study Timer with session, break, and streak tracking  
- 🎨 Field-Specific Tools:
  - Engineering: 100+ abbreviations, diagrams, curated YouTube channels  
  - Medical: Body system explorer, anatomy illustrations  
  - Arts: Digital drawing canvas with brush and color controls  
  - Commerce: Business calculator, financial terms  
  - Science: Scientific diagrams, terminology, and visual learning tools  
- 🎵 Spotify Integration for custom study music playlists  
- 💬 Motivational Quotes that rotate based on selected field  
- 📊 Progress Tracker and Visual Study Metrics  
- 🌙 Dark/Light Mode and fully responsive design

## 🧰 Built With  
- React 18  
- Vite

- Framer Motion  
- Zustand  
- TypeScript  
- LocalStorage API  
- Spotify & YouTube Embeds  
- Canvas API  

---

## 🧱 Challenges We Faced  
- Building a **persistent cross-page timer**  
- Designing unique yet unified layouts for **multiple fields**  
- Ensuring smooth performance with heavy content like videos and diagrams  
- Handling **Spotify API limitations** — currently, only embedded playlists work; direct playback is planned for the future
